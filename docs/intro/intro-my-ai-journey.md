

# About my AI journey

Good documentation doesn't happen by accident. It takes clear thinking, careful structure, and a real understanding of what readers need, and that's what I've been doing for years across API docs, user guides, and release notes.

Since 2025, I've been diving into how AI can make that process even better. I've tested tools, refined workflows, and learned firsthand where AI adds genuine value and where human curation matters most.

- Brainstorming and outlining

- Accelerating review and iteration cycles
- More to come as I explore agents and automation

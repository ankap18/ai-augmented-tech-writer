Manage synonym relationships for search and query expansion.

Use this API to define how search terms relate to each other. You can create
equivalent, one-way, or group-based synonym sets and retrieve them at query time
to improve search relevance, autocomplete, and query understanding.

Typical use cases include:
- Expanding user queries (for example, "laptop" → "notebook")
- Normalizing vocabulary across domains
- Managing controlled vocabularies or product taxonomies

Base URL: https://api.synonyms.io/v1
